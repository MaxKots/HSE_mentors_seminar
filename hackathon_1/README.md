Название Кейса: YourLook — AI-примерочная для онлайн-шопинга

Краткое описание кейса: YourLook — мобильное приложение, позволяющее пользователям примерять одежду из популярных интернет-магазинов прямо на себе с помощью искусственного интеллекта. Приложение помогает визуализировать, как конкретные вещи будут выглядеть на пользователе, облегчая выбор перед покупкой.

Готовый результат: 

Работающее мобильное приложение, позволяющее:

- Просматривать ассортимент магазинов одежды;
    
- Загружать свою фотографию или использовать виртуального манекена;
    
- Примерять выбранные вещи в один клик;
    
- Получать реалистичную визуализацию образа;
    
- Переходить по ссылкам на покупку товара.

Основной функционал:
1. Выбор интернет-магазина из доступных партнёров;
    
2. Просмотр каталога одежды с возможностью сохранения вещей в «примерочную»;
    
3. Примерка одежды на собственной фотографии;
    
4. Альтернатива — примерка на виртуальном манекене;
    
5. Прямой переход к покупке товара на сайте магазина.

Дополнительные требования
- Безопасное хранение и обработка пользовательских фотографий;
    
- Возможность ввода собственных параметров (рост, вес, обхват груди и пр.) для более точной примерки;
    
- Адаптация одежды под введённые размеры.
Пример использования:

Пользователь выбирает интересующий магазин и находит понравившуюся футболку. Он сохраняет её в личную примерочную, загружает своё фото (или использует манекен), и система генерирует изображение с примеркой выбранной вещи. После этого пользователь может сохранить образ, поделиться им или перейти к покупке.
